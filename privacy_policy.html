2. Nature de l'Application

OleaDiet est une application d'éducation nutritionnelle, de simulation et d'auto-évaluation. Elle n'est pas un dispositif médical au sens du Règlement (UE) 2017/745 (MDR) et ne remplace en aucun cas un avis médical professionnel, un diagnostic, une prescription ou une consultation clinique auprès d'un médecin ou d'un professionnel de santé qualifié.

3. Données Collectées et Finalités

Nous collectons et traitons les données suivantes aux fins décrites ci-après :

Catégorie de données Exemples Finalité Base légale (RGPD)
Profil utilisateur Sexe (M/F), tranche d'âge, objectifs nutritionnels Personnalisation de l'expérience utilisateur Consentement explicite (Art. 6.1.a)
Données nutritionnelles Repas enregistrés, aliments favoris, scores de compatibilité consultés Fonctionnalité de suivi alimentaire Consentement explicite (Art. 6.1.a)
Préférences d'application Langue, thème visuel, régimes favoris, calculateurs favoris Personnalisation de l'interface Intérêt légitime (Art. 6.1.f)

Note concernant la publicité : Les publicités sont actuellement désactivées dans l'application. Si elles sont activées dans une version ultérieure, le SDK AdMob de Google pourra collecter des identifiants techniques sous réserve de votre consentement explicite.

4. Stockage des Données — Architecture Offline-First

Vos données de santé et de nutrition restent sur votre appareil.

L'ensemble des données personnelles (profil, repas, favoris, préférences) est stocké exclusivement sur la mémoire interne de votre appareil mobile, via le moteur de base de données embarqué Isar. Aucune de vos données personnelles ne transite par un serveur distant ni n'est hébergée dans le cloud.

Les données ne sont pas chiffrées au repos sur l'appareil. La sécurité de vos données repose sur les mécanismes de sandboxing du système d'exploitation mobile (Android/iOS) qui isolent les données de l'application des autres applications du téléphone.

5. Partage avec des Tiers

OleaDiet ne transmet aucune donnée personnelle à des serveurs tiers pour traitement.

L'application n'utilise pas de serveurs distants ni d'API externes pour la gestion des repas ou du profil utilisateur.

6. Durée de Conservation

Vos données sont conservées sur votre appareil tant que l'application est installée. Vous pouvez les supprimer à tout moment :

  • Via le bouton « Supprimer toutes mes données » situé dans l'écran Paramètres de l'application.
  • En effaçant les données de l'application depuis les paramètres système de votre appareil.
  • En désinstallant l'application de votre appareil mobile.

7. Vos Droits (RGPD — Articles 15 à 22)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès (Art. 15) : Toutes vos données sont stockées localement et consultables directement dans l'application.
  • Droit de rectification (Art. 16) : Vous pouvez modifier vos données de profil à tout moment dans les Paramètres.
  • Droit à l'effacement (Art. 17) : Vous pouvez supprimer toutes vos données via le bouton dédié dans les Paramètres ou en désinstallant l'application.
  • Droit à la portabilité (Art. 20) : Vous pouvez exporter vos données au format JSON à tout moment depuis les Paramètres.

Pour toute demande relative à l'exercice de vos droits, vous pouvez nous contacter à : smofi.tech@gmail.com

8. Données des Mineurs

OleaDiet n'est pas destinée aux enfants de moins de 16 ans (ou l'âge minimum de consentement numérique applicable dans votre pays). Nous ne collectons pas sciemment de données relatives à des mineurs.
